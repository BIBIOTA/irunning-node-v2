#### Requires

- node.js (LTS)

#### How to install this service on local?

1. Install dependency packages.
```
npm install -g pnpm
pnpm install
```

2. Setting environment variable.
```
cp env.example .env
```

3. run dev
```
pnpm start:dev
```
